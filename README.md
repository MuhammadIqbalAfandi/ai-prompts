# 🤖 AI Prompts Collection

A curated collection of reusable AI prompts for software development, workflow automation, and vibe coding.

---

## 📌 Overview

This repository contains structured prompts designed to:

- Improve development workflow using AI
- Standardize output quality
- Enable fast and iterative "vibe coding"

---

## 🧠 Philosophy

- Keep prompts **simple and modular**
- Focus on **clear instructions + constraints**
- Optimize for **iteration and review**
- Avoid over-engineering unless needed

---

## 📂 Folder Structure

```md
ai-prompts/
├── 01-workflow/ # Step-by-step workflows (GitHub, coding, etc.)
├── 02-templates/ # Reusable prompt templates
├── 03-stacks/ # Stack-specific prompts (Go, Bun, etc.)
├── 04-usecases/ # Problem-based prompts (CRUD, auth, etc.)
├── 05-rules/ # Output rules & conventions
```

---

## 🚀 How to Use

1. Pick a prompt from the relevant folder
2. Replace variables (e.g. `{ISSUE_URL}`)
3. Run step-by-step (do not combine all prompts)
4. Review output before continuing

---

## 🧩 Prompt Format

Each prompt follows this structure:

```md
# Role (optional)

# Objective

# Inputs

# Tasks

# Output Requirements
```
